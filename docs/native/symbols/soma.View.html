Class soma.View

The View class is not dependant of the framework and is completely optional. Its purpose is to dispatch commands in an easier way. Commands can be dispatched from views and will not tight your views to the framework as they are simple native events (with a property bubbles set to true to be considered as command by the framework). The commmands must be dispatched from a DOM Element in order for the framework to catch them (see the examples).
