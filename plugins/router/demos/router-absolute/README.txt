The SomaRouter plugin is using the library Davis.js, more information there:
http://davisjs.com/

To run this demo on your local server, you must either:

1. create a virtual host
2. put the demo on the root of your server

You will get urls like:

http://localhost/
http://localhost/home
http://localhost/about
http://localhost/services
http://localhost/contact
http://localhost/articles
http://localhost/articles/article1
http://localhost/articles/article2
http://localhost/articles/article3
